# Guide Utilisateur - Générateur VCF et QR Code

## Informations Générales

**Objectif**: Créer des cartes de visite numériques au format VCF et générer des QR codes pour un partage instantané de vos informations de contact.

**Accès**: Public - Aucune connexion requise

## Propulsé par Manus

Cette application web moderne utilise une stack technologique de pointe pour offrir une expérience utilisateur fluide et performante. Le frontend est construit avec **React 19** et **TypeScript**, garantissant une interface réactive et typée. L'interface utilisateur s'appuie sur **Tailwind CSS 4** pour un design moderne et responsive, complété par **shadcn/ui** pour des composants élégants et accessibles. La génération de QR codes est assurée par la bibliothèque **qrcode**, offrant des codes de haute qualité compatibles avec tous les lecteurs standards.

**Déploiement**: Infrastructure auto-scalable avec CDN global pour des temps de chargement optimaux partout dans le monde.

## Utiliser Votre Site Web

L'application propose deux fonctionnalités principales accessibles via des onglets dédiés. Dans l'onglet "Formulaire VCF", vous pouvez créer votre carte de visite numérique en remplissant vos informations de contact. Saisissez votre nom complet, votre téléphone et votre email dans les champs obligatoires marqués d'un astérisque rouge. Vous pouvez également ajouter des informations optionnelles comme votre entreprise, titre du poste, adresse, site web et notes complémentaires. Une fois le formulaire complété, cliquez sur "Générer VCF" pour créer votre fichier de contact. Le bouton "Télécharger VCF" devient alors actif et vous permet de sauvegarder le fichier sur votre appareil.

L'onglet "QR Code" vous permet de transformer votre carte de visite en un code QR scannable. Après avoir généré un fichier VCF, basculez vers cet onglet et cliquez sur "Générer QR Code". Le code apparaît instantanément dans un cadre élégant avec bordure violette. Utilisez le bouton "Télécharger QR Code" pour sauvegarder l'image au format PNG. Vous pouvez également consulter l'aperçu du contenu VCF directement sous le QR code pour vérifier les informations encodées.

## Prochaines Étapes

Discutez avec Manus AI à tout moment pour demander des modifications ou ajouter des fonctionnalités. Commencez dès maintenant en remplissant le formulaire avec vos informations de contact pour créer votre première carte de visite numérique et son QR code associé.
